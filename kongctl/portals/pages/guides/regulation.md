# 📋 利用規約とポリシー

Jungle Store APIを安全かつ適切に利用するための規約とポリシーをご確認ください。

::alert
---
type: "warning"
---
**重要:** APIを使用することで、これらの規約に同意したものとみなされます。
::

---

## 📜 利用規約

### 1. アカウントとアクセス

#### 1.1 アカウント登録

- 正確かつ完全な情報を提供すること
- アカウント情報を最新の状態に保つこと
- 1人につき1つのアカウントのみ作成可能

#### 1.2 API Keyの管理

**✅ 許可される使用**

- セキュアな環境でのAPI Keyの保管
- 環境変数や秘密管理システムの使用
- 定期的なKeyのローテーション

**❌ 禁止される使用**

- API Keyの公開リポジトリへのコミット
- クライアントサイドコードへの埋め込み
- 第三者への無断譲渡

---

### 2. 利用制限

#### 2.1 レート制限

各APIのレート制限を遵守すること：

| スコープ | リクエスト/分 |
|----------|-------------|
| グローバル | 60 |
| Order API | 10 |

> 💡 **注意:** レート制限を意図的に回避する行為は禁止されています。

#### 2.2 公正な利用

以下の行為は禁止されています：

- **過度なリクエスト**: 必要以上に頻繁なAPI呼び出し
- **リソースの独占**: 他のユーザーに影響を与える使用
- **スクレイピング**: APIの目的外利用
- **DDoS攻撃**: 意図的なサービス妨害

---

## 🔒 プライバシーポリシー

### データ収集

#### 収集する情報

- アカウント情報（氏名、メールアドレス）
- APIリクエストログ（IPアドレス、タイムスタンプ、エンドポイント）
- パフォーマンスメトリクス

### データの使用目的

- **サービス提供**: API機能の提供と維持
- **サポート**: 技術サポートとトラブルシューティング
- **改善**: サービス品質の向上
- **セキュリティ**: 不正利用の検知と防止

### データの保護

| 項目 | 対策 |
|------|------|
| 🔐 **暗号化** | 転送時: TLS 1.3 / 保管時: AES-256 |
| 🏢 **アクセス制御** | 最小権限の原則 |
| 🔍 **監査** | 定期的なセキュリティ監査 |

### データの保持期間

| データ種別 | 保持期間 |
|-----------|---------|
| アカウント情報 | アカウント削除まで |
| APIリクエストログ | 90日間 |
| エラーログ | 30日間 |

### ユーザーの権利

GDPR、CCPAに基づく権利：

- **アクセス権**: 保存データの開示請求
- **修正権**: 不正確なデータの修正
- **削除権**: データ削除の要求（Right to be forgotten）
- **移行権**: データのポータビリティ

- APIの利用権（非独占的、譲渡不可）

## 🌍 コンプライアンス

### 地域規制

#### GDPR（EU一般データ保護規則）

- データ処理の合法性確保
- データ主体の権利保護
- データ侵害時の72時間以内通知

#### 個人情報保護法（日本）

- 個人情報の適切な取り扱い
- 安全管理措置の実施
- 本人同意の取得

### セキュリティ対応

#### 脆弱性報告

セキュリティ脆弱性を発見した場合：

1. **報告先**: セキュリティチームへ連絡
2. **情報提供**: 再現手順、影響範囲
3. **対応**: 24時間以内に初期応答

---

## 🚫 禁止事項

以下の行為は明示的に禁止されています：

### 技術的禁止事項

- **リバースエンジニアリング**: APIの逆アセンブル、逆コンパイル
- **セキュリティ回避**: 認証メカニズムの回避
- **負荷テスト**: 事前承認なしの負荷テスト
- **自動化**: レート制限を超える自動化スクリプト

### 利用目的の禁止事項

- **違法行為**: 法律に違反する用途
- **スパム**: 迷惑メール、スパムの送信
- **詐欺**: 詐欺的な活動

---

## ⚠️ 違反時の対応

### 警告プロセス

1. **第1段階**: 警告メールの送信
2. **第2段階**: アカウント一時停止（7日間）
3. **第3段階**: アカウント永久停止

### 即時停止対象

以下の場合は警告なしに停止：

- 重大なセキュリティ違反
- 違法行為
- DDoS攻撃

---

## 📝 規約の変更

- 軽微な変更: 即時発効
- 重大な変更: 30日後に発効
- 変更後もAPIを継続使用することで、新しい規約への同意とみなされます
