## 事前準備

すべてのシナリオ共通の事前準備です。

```bash
# 全サービスを起動
docker compose up -d --build

# 全コンテナの起動を確認
docker compose ps
```

デモユーザー情報:

| 名前                 | メール              | パスワード    | API Key         |
| -------------------- | ------------------- | ------------- | --------------- |
| ゴリラ太郎           | `user@example.com`  | `password123` | `demo-api-key`  |
| シルバーバック管理者 | `admin@example.com` | `password123` | `admin-api-key` |

詳しい起動手順は [はじめに](../getting-started.md) を参照してください。
